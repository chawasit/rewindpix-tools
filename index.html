RewindPix Tools

Browser tools for the RewindPix camera (Novatek NT96565) over its local WiFi API. No install — just open a tool while joined to the camera's WiFi.

Set roll size →

Set the frame budget (max photos) to any number. Default 99; 0 clears the roll.
Run these locally. The camera is plain HTTP (192.168.1.254), so a tool must open over file:// or http:// — not HTTPS (browsers block HTTPS→HTTP requests, which is why this isn't on GitHub Pages). Download a tool and open the file, serve the folder over http (python -m http.server), or even serve it from the camera's own SD card at http://192.168.1.254/… (same-origin — no block).
Camera API notes: RewindPix / NT96565 WiFi API. Unofficial · reverse-engineered · use at your own risk.
