Set roll size

Sets the camera's frame budget (max photos) via cmd=8004. Any non-negative number; 0 clears the roll.

  1. Open this page (you're here).
  2. Connect your phone / PC to the camera's WiFi.
  3. Enter a number and click "Set roll size".
  4. Wait for ✓ success (green) or an error (red) below.
  5. Disconnect from the camera WiFi and use the camera as normal.
This page is loaded over HTTPS, but the camera speaks plain HTTP. Your browser will block the request. Open this tool from a local file or over http://, or allow “Insecure content” for this site in your browser’s site settings.
How it works & troubleshooting

Sends GET <base>/?custom=1&cmd=8004&par=<N> to the camera. It's fire-and-forget (the camera doesn't send CORS headers, so the reply can't be read) — verify the new count on the camera's screen.

  • Join your device to the camera's WiFi first; the camera must be reachable at the base URL.
  • An SD card must be inserted for photos; the frame budget still sets without one.
  • 99 is the highest value confirmed to be accepted; larger values are untested.
  • Single-client device — don't hammer it; one action at a time.

API details: see the RewindPix camera API notes.
